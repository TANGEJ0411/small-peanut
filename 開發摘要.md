# 育嬰紀錄 Web App 開發規劃書

## 1. 專案願景

開發一個專供個人使用、以**行動端優先 (Mobile First)** 為核心的育嬰紀錄工具。強調「單手操作」的便利性，協助家長精準掌握寶寶作息並轉化為數據分析。

---

## 2. 功能清單

### Phase 1 — 核心 CRUD（待開發）

| 功能 | 說明 |
|---|---|
| 餵食紀錄 | 母奶親餵（時長 + 左/右側切換）、擠奶量、配方奶量、副食品 |
| 睡眠紀錄 | 入睡/醒來時間，後端自動計算單次及全天總睡眠時長 |
| 排泄紀錄 | 乾淨/尿尿/便便，記錄顏色與質地 |
| 成長指標 | 身高、體重、頭圍的定期錄入 |
| 快捷狀態標籤 | 開心、哭鬧、發燒、長牙、猛長期 |
| Today Dashboard | 首頁快覽：最後餵奶/換尿布時間、今日總睡眠、預計下次餵奶 |
| 暗色模式 | 夜間哺乳情境必要，與 Phase 1 同步實作 |

### Phase 2 — 分析與體驗（待開發）

| 功能 | 說明 |
|---|---|
| 計時器 | 親餵/睡眠計時，結束後自動帶入紀錄表單 |
| 作息甘特圖 | 24 小時作息分佈視覺化 |
| WHO 成長曲線 | 身高/體重/頭圍對比 WHO 百分位標準 |
| PWA 支援 | 安裝至手機桌面、離線快取 |
| 多人即時同步 | SSE 推播，供夫妻/長輩共用同一份資料 |
| 健康管理 | 用藥紀錄、疫苗施打清單、就醫日誌 |
| AI 週摘要 | 分析一週趨勢，提供健康摘要建議 |

### Phase 3 — 本階段不做

JWT 安全驗證、CSV/PDF 匯出、S3 照片儲存已確認排除。

## 3. 技術棧

| 層級 | 技術 | 備註 |
|---|---|---|
| Backend | Spring Boot 3.4.5 + Java 21 | 虛擬執行緒 (Virtual Threads) |
| Frontend | React 19 + Tailwind CSS 3 | 暗色模式用 `dark:` class |
| Build | Gradle | bootJar 含前端打包 |
| Database | MySQL 8.0 | ddl-auto=update，不需手動建表 |
| 圖表 | Recharts | 作息甘特圖、成長曲線 |
| 即時同步 | Server-Sent Events (SSE) | 低頻更新場景，比 WebSocket 輕量 |

## 5. 設計原則

- **單手操作優先**：互動元素最小 44×44px，大按鈕設計
- **輸入效率第一**：表單時間預設現在，計時器結束自動帶入，減少手動輸入
- **後端計算時長**：前端只傳時間點，睡眠/餵奶時長在 Java 21 計算後回傳
- **時間統一存 UTC**：前端顯示時才轉換本地時區
- **API 回應精簡**：避免冗餘欄位，降低等待感
